My “Extra” Feature

I learned how to make sparkles like the ones that you can see in the background, using CSS and javascript.

"position: fixed" so the sparkles stay in place regardless of scrolling
"bottom: 0" so it starts at the bottom
"width: 6px" / "height: 6px" to determine size
"border-radius:50%" to make them round
"background: radial-gradient(circle, #ffb0e9 0%, transparent 70%)" gives it the glowing effect by making the inner circle a brighter pink and slowly fading as it gets farther away from the center
"animation: rise linear infnite" references the @keyframes block named rise, tells the sparkles to move at a constant speed, and to do so infinitely, restarting once it reaches the end
"pointer-events: none" prevents the sparkles from being clicked so they don't interfere on the page
"opacity: 0.8" makes the sparkles slightly see-through

"0%{ transform: translateY(0) scale(0.5)" at the beginning of the animation, start at the bottom at half normal size
"opacity: 1" starts fully visible
"80% { opacity: 1" prevents fade until 80% way through the animation
"100% { transform: translateY(-110vh) scale(1)" defines the end as 110% the height of the screen. It will grow to its full scale(1) by the end.
"opacity: 0" by the end, it should be fully transparent

"document.addEventListener('DOMContentLoaded', function() {" when the content of the page has loaded, run the function
"for (let i = 0; i < 60; i++) {" loops 60 times (creates 60 sparkles)
"const s = document.createElement('div')" creates a div element in memory called 's'
"s.className = 'sparkle'" gives the 's' element the attributes of the 'sparkle' class
"s.style.left = Math.random() * 100 + 'vw'" generates a random number between 1 and 100 that determines how far from the left each sparkle will start (i.e. 60% = 60% of the screen width from the left)
"s.style.animationDelay = Math.random() * 10 + 's'" sets a random wait time between 1 and 10 seconds before the animation starts
"s.style.animationDuration = (8 + Math.random() * 6) + 's'" determines the sparkles speed by generating a random number from 0-6 + 8
"document.body.appendChild(s)" attaches the sparkle to the body of the page
